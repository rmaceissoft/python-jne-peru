============
Installation
============


Stable release
--------------

To install Python JNE Peru, run this command in your terminal:

.. code-block:: console

    $ pip install pyjne_peru

This is the preferred method to install Python JNE Peru, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Python JNE Peru can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/rmaceissoft/pyjne_peru

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/rmaceissoft/pyjne_peru/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/rmaceissoft/pyjne_peru
.. _tarball: https://github.com/rmaceissoft/pyjne_peru/tarball/master
